# Core Python Topics

The following topics are typically included in a **beginner-level Python programming syllabus**.

## 1. Introduction to Python
- What is Python and its history
- Installing Python and setting up the development environment  
  - IDLE  
  - Anaconda  
  - VS Code
- Writing and running your first program (`Hello, World!`)
- Keywords, identifiers, syntax, and indentation

### Variables, Data Types, Operators and Expressions
- Declaration and use of variables
- Core data types:
  - Integers (`int`)
  - Floating-point numbers (`float`)
  - Strings (`str`)
  - Booleans (`bool`)
- Type conversion (typecasting)
- User input and console output
#### Operators 
- Arithmetic operators
- Assignment operators
- Comparison operators
- Logical operators
- Identity operators
- Membership operators
- Operator precedence

### Lists
- Creating lists
- Accessing and modifying elements
- Looping through lists

### Tuples
- Understanding immutable ordered sequences

### Dictionaries
- Working with key-value pairs

### Sets
- Understanding unordered collections of unique elements

### Strings
- String slicing
- String methods
- String formatting

- Conditional execution using:
  - `if`
  - `else`
  - `elif`
- Nested conditionals
- `for` loops
- `while` loops
- Using the `range()` function
- Nested loops
- `break`, `continue`, and `pass` statements

## 3. Functions and Modules

  - Positional arguments

- Importing and using built-in and external modules/packages
